Multi-density asset generator for Android.

## Usage

1. **Upload** - Drag & drop PNG/JPG/WebP images
2. **Configure** - Set input scale (1x-4x), WebP quality, and output densities
3. **Download** - Get a ZIP with organized drawable folders

## Development

```bash
# Install
npm install

# Dev server
npm run dev

# Build
npm run build
```
